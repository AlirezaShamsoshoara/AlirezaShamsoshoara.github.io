## [AI Daily Paper Summary](/blog/ai-papers/)

I read AI and ML research papers daily and post concise summaries here. Each entry covers the key contributions, methods, and takeaways from recent papers across topics like deep learning, reinforcement learning, NLP, computer vision, and more.

<!-- LATEST_PAPERS_START -->
**Latest — May 30, 2026**

1. **Alignment Tampering: How RLHF Is Exploited to Optimize Misaligned Biases** — LLMs being RLHF-trained can corrupt their own preference dataset, causing alignment to amplify undesired behaviors rather than suppress them. [arxiv](https://arxiv.org/abs/2605.27355)
2. **SkillOpt: Executive Strategy for Self-Evolving Agent Skills** — Treats agent skills as external trainable states optimized iteratively under feedback, reliably improving agent performance from its starting point. [arxiv](https://arxiv.org/abs/2605.23904)
3. **CONF-KV: Confidence-Aware KV Cache Eviction with Mixed-Precision Storage for Long-Horizon LLM** — Uses model confidence signals to drive dynamic KV-cache eviction and mixed-precision storage, cutting memory pressure on long-context tasks. [arxiv](https://arxiv.org/abs/2605.24786)
4. **PANDO: Efficient Multimodal AI Agents via Online Skill Distillation** — Web agents get cheaper as they accumulate experience by distilling frontier model trajectories into compact lightweight skills. [arxiv](https://arxiv.org/abs/2605.24785)
5. **ARIS: Autonomous Research via Adversarial Multi-Agent Collaboration** — Open-source harness where adversarial LLM agents conduct end-to-end research including hypothesis generation, experiment design, and result analysis. [arxiv](https://arxiv.org/abs/2605.03042)
6. **DynaFLIP: Rethinking Robotics Perception via Tri-Modal-Dynamics Guided Representation** — Integrates RGB, depth, and event-camera modalities under a dynamics-guided pre-training objective, encoding action-relevant temporal dynamics for robot manipulation. [arxiv](https://arxiv.org/abs/2605.30350)
7. **Thinking Before Constraining: A Unified Decoding Framework for Large Language Models** — "In-Writing" decoding lets LLMs reason freely before applying structural constraints, preserving quality while ensuring output verifiability in a single pass. [arxiv](https://arxiv.org/abs/2601.07525)
8. **Mem0: Building Production-Ready AI Agents with Scalable Long-Term Memory** — Scalable memory-centric architecture that dynamically extracts and retrieves salient information across multi-session dialogues without full context replay. [arxiv](https://arxiv.org/abs/2504.19413)
9. **RUBRIC-ARROW: Alternating Pointwise Rubric Reward Modeling for LLM Post-training in Non-verifiable Domains** — Decomposes reward modeling into explicit rubric criteria with alternating pointwise scoring, enabling reliable alignment in creative and open-ended domains. [arxiv](https://arxiv.org/abs/2605.29156)
10. **REPOT: Recoverable Program-of-Thought via Checkpoint Repair** — Adds deterministic checkpoint replay to PoT — on failure, resumes from the verified prefix with a single LLM call instead of regenerating from scratch. [arxiv](https://arxiv.org/abs/2605.30052)
