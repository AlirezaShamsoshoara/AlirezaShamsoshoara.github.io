## [AI Daily Paper Summary](/blog/ai-papers/)

I read AI and ML research papers daily and post concise summaries here. Each entry covers the key contributions, methods, and takeaways from recent papers across topics like deep learning, reinforcement learning, NLP, computer vision, and more.

**Latest — June 07, 2026**

1. **SABER** — Benchmark for LLM coding agent operational safety in stateful multi-file workspaces, measuring cascading harm beyond single-shot refusals. [arxiv](https://arxiv.org/abs/2606.01317)
2. **Code2LoRA** — Hypernetwork that instantly generates repo-specific LoRA adapters from code files, eliminating per-repo fine-tuning and stale RAG context. [arxiv](https://arxiv.org/abs/2606.06492)
3. **LLM Anonymization Against Agentic Re-Identification** — Defense against web-search-equipped agents that can cross-reference weak cues to re-identify anonymized individuals. [arxiv](https://arxiv.org/abs/2605.30848)
4. **WLA (World-Language-Action)** — Unified embodied foundation model jointly predicting subtasks, subgoal images, and robot actions from text, images, and robot states. [arxiv](https://arxiv.org/abs/2606.05979)
5. **PropMe** — Propensity-aware memorization framework distinguishing a model's *capability* to leak training data from its *propensity* to do so in normal use. [arxiv](https://arxiv.org/abs/2606.06286)
6. **Benchmark Everything Everywhere All at Once** — Automated framework generating diverse, multi-dimensional benchmarks to combat saturation and contamination in LLM evaluation. [arxiv](https://arxiv.org/abs/2606.06462)
7. **TIDE** — Proactive multi-problem discovery agent that scans codebases and documents to surface hidden issues without explicit user direction. [arxiv](https://arxiv.org/abs/2606.04743)
8. **Meta-Cognitive Memory Policy Optimization** — Intermediate memory quality signals fix progressive summarization degradation in long-horizon LLM agents. [arxiv](https://arxiv.org/abs/2605.30159)
9. **EvoDS** — Self-evolving data science agent with a growing skill library and selective memory compression for multi-stage analytical pipelines. [arxiv](https://arxiv.org/abs/2606.03841)
10. **Dream.exe** — Tests whether video generation models produce physically executable robot manipulation plans, revealing a systematic realism-vs-executability gap. [arxiv](https://arxiv.org/abs/2606.04811)
