## [AI Daily Paper Summary](/blog/ai-papers/)

I read AI and ML research papers daily and post concise summaries here. Each entry covers the key contributions, methods, and takeaways from recent papers across topics like deep learning, reinforcement learning, NLP, computer vision, and more.

**Latest — June 26, 2026**

1. **DanceOPD** — On-policy generative field distillation that unifies T2I, local editing, and global editing in one flow-matching model without capability conflicts. [arxiv.org/abs/2606.27377](https://arxiv.org/abs/2606.27377)
2. **ViQ** — Visual quantization framework matching continuous encoders on multimodal tasks with 20–70% training speedup via text-aligned discrete representations. [arxiv.org/abs/2606.27313](https://arxiv.org/abs/2606.27313)
3. **OPID** — On-policy skill distillation that extracts dense hindsight supervision from completed agent trajectories, solving the sparse-reward bottleneck in agentic RL. [arxiv.org/abs/2606.26790](https://arxiv.org/abs/2606.26790)
4. **Qwen-Image-Agent** — Agentic pipeline bridging the "Context Gap" in T2I generation via planning, web search, and memory to enrich underspecified user prompts. [arxiv.org/abs/2606.26907](https://arxiv.org/abs/2606.26907)
5. **The Verification Horizon** — Systematic analysis of Goodhart's Law in coding-agent reward signals; argues verification must co-evolve with generative capabilities. [arxiv.org/abs/2606.26300](https://arxiv.org/abs/2606.26300)
6. **JetFlow** — Breaks speculative decoding's scaling ceiling via parallel tree drafting, eliminating the causality-efficiency dilemma of autoregressive draft heads. [arxiv.org/abs/2606.18394](https://arxiv.org/abs/2606.18394)
7. **Tool-Use RL Collapse** — Traces catastrophic collapse in multi-step tool-use RL to control-token probability spikes; fix is interleaved SFT + RL training. [arxiv.org/abs/2606.26027](https://arxiv.org/abs/2606.26027)
8. **Fast LeWorldModel** — Replaces autoregressive world-model rollout with parallel action-prefix prediction, cutting cost and error accumulation for long-horizon planning. [arxiv.org/abs/2606.26217](https://arxiv.org/abs/2606.26217)
9. **GauntletBench** — Web-based agent benchmark exposing gaps in temporal perception, graphical understanding, and 3D reasoning invisible in saturated existing evals. [arxiv.org/abs/2606.14397](https://arxiv.org/abs/2606.14397)
10. **LISA** — Likelihood score alignment reveals ControlNet side branches contribute likelihood scores, not features; regularization improves controllable generation efficiency. [arxiv.org/abs/2606.27192](https://arxiv.org/abs/2606.27192)
