## [AI Daily Paper Summary](/blog/ai-papers/)

I read AI and ML research papers daily and post concise summaries here. Each entry covers the key contributions, methods, and takeaways from recent papers across topics like deep learning, reinforcement learning, NLP, computer vision, and more.

**Latest — June 27, 2026**

1. **Autoregressive Boltzmann Generators** — Replaces normalizing flows in Boltzmann Generators with autoregressive models, overcoming the expressivity bottleneck in equilibrium molecular sampling. [arxiv.org/abs/2606.27361](https://arxiv.org/abs/2606.27361)
2. **Progress Advantage for LLM Agents** — The log-prob ratio between RL policy and reference policy is a free, annotation-free step-level reward signal that beats trained process reward models. [arxiv.org/abs/2606.26080](https://arxiv.org/abs/2606.26080)
3. **RiVER** — Trains LLMs via group-relative RL on continuous execution feedback, enabling policy improvement on open-ended tasks without ground-truth answers. [arxiv.org/abs/2606.27369](https://arxiv.org/abs/2606.27369)
4. **CARVE** — Fixes three fundamental defects in delta-rule recurrent architectures with content-aware memory gates and chunk-parallel execution. [arxiv.org/abs/2606.27229](https://arxiv.org/abs/2606.27229)
5. **BINEVAL** — Decomposes LLM evaluation into atomic binary questions for transparent, multi-dimensional quality scores instead of opaque holistic judgments. [arxiv.org/abs/2606.27226](https://arxiv.org/abs/2606.27226)
6. **E-TTS** — Test-time scaling framework for robotic manipulation that preserves historical context across long-horizon sequential tasks. [arxiv.org/abs/2606.27268](https://arxiv.org/abs/2606.27268)
7. **Ask, Solve, Generate** — Self-evolving multimodal framework with Proposer/Solver/Generator roles that jointly improve visual understanding and generation from unlabeled images. [arxiv.org/abs/2606.27376](https://arxiv.org/abs/2606.27376)
8. **GUI Agents via Experience Exploration** — Autonomous exploration + hindsight utilization enables small open-source MLLMs to generalize across web tasks without commercial APIs. [arxiv.org/abs/2606.27330](https://arxiv.org/abs/2606.27330)
9. **Omnimodal Embodied Agents** — Unified cyber-physical action space spanning APIs, IoT, manipulation, and navigation with autonomous recovery for long-horizon operation. [arxiv.org/abs/2606.27251](https://arxiv.org/abs/2606.27251)
10. **Sparse Autoencoder Sparsity Regularizers** — Soft sparsity regularizers replace Top-k hard constraints in SAEs, enabling more principled and interpretable feature decompositions. [arxiv.org/abs/2606.27321](https://arxiv.org/abs/2606.27321)
