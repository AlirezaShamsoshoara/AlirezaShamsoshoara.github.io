## [AI Daily Paper Summary](/blog/ai-papers/)

I read AI and ML research papers daily and post concise summaries here. Each entry covers the key contributions, methods, and takeaways from recent papers across topics like deep learning, reinforcement learning, NLP, computer vision, and more.

**Latest — May 29, 2026**

1. **AgentDoG 1.5: A Lightweight and Scalable Alignment Framework for AI Agent Safety and Security** — Taxonomy-guided training with influence-function purification to align open-world agents against emergent safety and security threats. [arxiv](https://arxiv.org/abs/2605.29801)
2. **Qwen-VLA: Unifying Vision-Language-Action Modeling across Tasks, Environments, and Robot Embodiments** — DiT-based action decoder with embodiment-aware prompt conditioning unifies heterogeneous robot decision-making in a single foundation model. [arxiv](https://arxiv.org/abs/2605.30280)
3. **OmniRetrieval: Unified Retrieval across Heterogeneous Knowledge Sources** — Identifies appropriate knowledge repositories and dispatches source-native queries to their execution engines, outperforming single-source RAG retrievers. [arxiv](https://arxiv.org/abs/2605.29250)
4. **CollectionLoRA: Collecting 50 Effects in 1 LoRA via Multi-Teacher On-Policy Distillation** — Probabilistic Dual-Stream Routing compresses up to 50 visual effects into a single LoRA for efficient deployment without quality loss. [arxiv](https://arxiv.org/abs/2605.25378)
5. **minWM: A Full-Stack Open-Source Framework for Real-Time Interactive Video World Models** — Complete open-source pipeline from data construction to streaming inference, enabling low-latency interactive video world models. [arxiv](https://arxiv.org/abs/2605.30263)
6. **YoCausal: How Far is Video Generation from World Model? A Causality Perspective** — VoE-inspired benchmark using temporally reversed real-world videos to measure causal cognition vs. statistical pattern matching in video models. [arxiv](https://arxiv.org/abs/2605.30346)
7. **GenClaw: Code-Driven Agentic Image Generation** — Three-stage code-driven framework lets LLMs directly control image construction via programmatic logic instead of repetitive prompt rewriting. [arxiv](https://arxiv.org/abs/2605.30248)
8. **EarlyTom: Early Token Compression Completes Fast Video Understanding** — Training-free framework that compresses visual tokens inside the vision encoder, reducing time-to-first-token for video LLMs. [arxiv](https://arxiv.org/abs/2605.30010)
9. **UniSteer: Text-Guided Flow Matching in Activation Space for Versatile LLM Steering** — Universal conditional velocity field in activation space enables flexible LLM behavior steering from natural language descriptions alone. [arxiv](https://arxiv.org/abs/2605.30076)
10. **How LoRA Remembers? A Parametric Memory Law for LLM Finetuning** — Establishes a power-law between LoRA rank and parametric memory capacity with a threshold-guided optimization framework for continual finetuning. [arxiv](https://arxiv.org/abs/2605.30260)
