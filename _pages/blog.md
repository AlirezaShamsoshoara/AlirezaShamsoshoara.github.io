## [AI Daily Paper Summary](/blog/ai-papers/)

I read AI and ML research papers daily and post concise summaries here. Each entry covers the key contributions, methods, and takeaways from recent papers across topics like deep learning, reinforcement learning, NLP, computer vision, and more.

<!-- LATEST_PAPERS_START -->
**Latest — June 01, 2026**

1. **SANA-Streaming: Real-time Streaming Video Editing with Hybrid Diffusion Transformer** — NVIDIA's system-algorithm co-designed framework for high-resolution, real-time streaming V2V editing on consumer GPUs via a Hybrid Diffusion Transformer. [arxiv](https://arxiv.org/abs/2605.30409)
2. **GrepSeek: Training Search Agents for Direct Corpus Interaction** — LLM search agents that bypass pre-built indices by interacting directly with raw corpora, showing strong gains on knowledge-intensive benchmarks. [arxiv](https://arxiv.org/abs/2605.29307)
3. **Mellum2 Technical Report** — JetBrains' open-weight 12B MoE model (2.5B active params) purpose-built for code generation, debugging, and agentic IDE tasks. [arxiv](https://arxiv.org/abs/2605.31268)
4. **VLM3: Vision Language Models Are Native 3D Learners** — Demonstrates that standard VLMs become capable 3D learners with focused training signals, eliminating need for specialist 3D architectures. CVPR 2026. [arxiv](https://arxiv.org/abs/2605.30561)
5. **Representation Forcing for Bottleneck-Free Unified Multimodal Models** — Removes the frozen VAE structural bottleneck in unified perception+generation models by forcing high-level representation alignment during training. [arxiv](https://arxiv.org/abs/2605.31604)
6. **dMoE: dLLMs with Learnable Block Experts** — Block-level MoE routing that resolves the mismatch between Diffusion LLMs' parallel decoding and token-level MoE expert assignment. [arxiv](https://arxiv.org/abs/2605.30876)
7. **LongTraceRL: Learning Long-Context Reasoning from Search Agent Trajectories with Rubric Rewards** — Trains LLMs on search agent trajectories with dense rubric-based intermediate rewards, improving long-context reasoning over sparse outcome-only signals. [arxiv](https://arxiv.org/abs/2605.31584)
8. **SCOPE: Self-Play via Co-Evolving Policies for Open-Ended Tasks** — Data-free self-play framework co-evolving a Challenger and Solver policy for open-ended LLM tasks without rule-checkable answers. [arxiv](https://arxiv.org/abs/2605.31433)
9. **SAAS: Self-Aware RL for Over-Search Mitigation in Agentic Search** — RL-trained self-awareness that teaches agents to skip external searches when internal knowledge suffices, reducing redundant retrieval calls. [arxiv](https://arxiv.org/abs/2605.29796)
10. **COLLEAGUE.SKILL: Automated AI Skill Generation via Expert Knowledge Distillation** — Automatically distills actionable expertise from heterogeneous human traces into compact, reusable agent skills without hand-authored skill libraries. [arxiv](https://arxiv.org/abs/2605.31264)
